# IoT Quiz - 50 Random Questions

An interactive online quiz covering **Internet of Things (IoT)** concepts and technologies. Built with HTML, CSS, and JavaScript.

## 🚀 Features

- **50 Random Questions** extracted from IoT course materials
- **Interactive Web Interface** - Answer questions in a modern, user-friendly interface
- **Progress Tracking** - Visual progress bar shows your completion status
- **Answer Review** - Review all your answers before submitting
- **Download Responses** - Export your answers as a text file
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **No Backend Required** - Runs entirely in the browser (no server needed)

## 📋 Topics Covered

- Smart Grid and Energy Management
- IoT Architecture and Components
- Wireless Technologies (WiFi, Bluetooth, LoRaWAN, etc.)

## 📖 How to Use

1. Open the quiz in your web browser
2. Read each question carefully
3. Type your answer in the text field
4. Click "Next" to proceed to the next question
5. Use "Previous" to review previous answers
6. Click "Submit Quiz" on the last question to finish
7. Review your answers and download them as a file if needed

## 🛠️ Installation

### Option 1: GitHub Pages (Recommended)

1. Fork or clone this repository
2. Go to **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Select **main** (or **master**) branch and **root** folder
5. Save - your quiz will be live in a few minutes!

### Option 2: Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/narutonamikaze/iot-quiz.git
   cd iot-quiz
   ```

2. Open `index.html` in your web browser
